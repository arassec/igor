# Connector Editor

The connector editor is used to configure connectors.
It is split into two parts.

![connector editor](./images/connector-editor.png)

## Connector Configuration

The connector configuration on the left side of the page is used to configure the connector.

![connector editor configuration](./images/connector-editor-navigator.png)

The top element contains buttons for the following actions:

![leave button](./images/leave-button.png) Navigates back to the job configuration from which the connector editor was opened.

![test button](./images/test-button.png) Tests the connector configuration and displays the results.

![save button](./images/save-button.png) Saves the connector.

Below the top element is the list of jobs that use this connector.
By clicking on an element in the list you can jump directly into the job editor for the corresponding job. 
 
## Parameters Editor

The parameters editor in the middle of the page contains all configuration parameters for the selected connector.

![connector editor parameters](./images/connector-editor-configurator.png)

By selecting different categories and types, the configuration parameters will change accordingly.

The ![help](./images/help-button.png) button can be used to display the online help for the selected connector.
