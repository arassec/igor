Igor Test Environment

Web

HTTP(S)

A local NGINX server is available providing the following test environment:
  • HTML test file (http://localhost/igor-web.html)
  • JSON test file (http://localhost/igor-web.json)
  • TXT test file (http://localhost/igor-web.txt)
HTTP Basic Authentication can be tested with the following URLs:
  • HTML test file (http://localhost/auth/igor-web.html)
  • JSON test file (http://localhost/auth/igor-web.json)
  • TXT test file (http://localhost/auth/igor-web.txt)
Redirects to HTTPS can be tested with the following links
  • HTML test file (http://localhost/redirect/igor-web.html)
  • JSON test file (http://localhost/redirect/igor-web.json)
  • TXT test file (http://localhost/redirect/igor-web.txt)
The files are also served via HTTPS:
  • HTML test file (https://localhost/igor-web.html)
  • JSON test file (https://localhost/igor-web.json)
  • TXT test file (https://localhost/igor-web.txt)
