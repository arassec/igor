# Connector Overview

The connector overview displays all configured connectors. 
Connectors can be managed here, e.g. created, duplicated or exported.

![connector overview](./images/connector-overview.png)

## Action Bar

On top of the page is the overview's action bar. 

![connector overview action bar](./images/connector-overview-action-bar.png)

On the left is a filter element:
- **Name Filter:** The name filter can be used to filter the displayed connectors by their name.

On the right part of the action bar are buttons to perform the following actions:

![add connector button](./images/add-connector-button.png) Opens the connector editor to create a new connector.

![import connector](./images/import-connector-button.png) Opens the dialog for importing connectors.

## Connector Tiles

The main element of the connector overview are the connector tiles.
Each connector is displayed in a separate tile.
Clicking on the tile opens the connector editor for the corresponding connector.

![connector-tile](./images/connector-overview-tile.png)

The tile contains the following buttons to modify the connector:

![delete job button](./images/delete-button.png) Deletes the connector.

![export job](./images/export-button.png) Exports the connector as JSON text file.

![duplicate job](./images/duplicate-button.png) Opens the connector editor with a duplicated connector configuration.
